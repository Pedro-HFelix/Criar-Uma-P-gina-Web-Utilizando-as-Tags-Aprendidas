HTML

Drescrição
VOlTAR
HTML (Hypertext Markup Language) é a linguagem de marcação padrão usada para criar conteúdo na web. Ele fornece uma estrutura básica para uma página da web , permitindo que os desenvolvedores adicionem texto, imagens, vídeos, links e outros elementos interativos. Ao usar tags HTML, os desenvolvedores podem definir a estrutura e a organização do conteúdo da página, bem como suas características visuais. O HTML é uma linguagem fundamental para qualquer desenvolvedor web, já que é a base para a construção de qualquer página da web.

CSS

Drescrição
VOlTAR
CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a apresentação e o layout do conteúdo em uma página da web. Ele permite que os desenvolvedores alterem a cor, fonte, tamanho, espaço entre linhas, posicionamento e outros aspectos visuais de elementos HTML. Com o CSS, é possível separar o conteúdo da página da sua apresentação visual, o que ajuda a manter o código organizado e torna mais fácil fazer alterações no design da página. Além disso, o CSS é capaz de tornar a página da web responsiva, adaptando-se automaticamente a diferentes tamanhos de tela, desde desktops até dispositivos móveis.

JAVASCRIPT

Drescrição
VOlTAR
JavaScript é uma linguagem de programação usada para adicionar interatividade e dinamismo a páginas da web. Ele permite que os desenvolvedores criem funcionalidades complexas, como animações, validação de formulários, menus interativos e outras interações dinâmicas que melhoram a experiência do usuário. Além disso, o JavaScript é usado para interagir com o DOM (Document Object Model), que é uma representação da estrutura da página na memória do computador. Com isso, é possível manipular elementos HTML e alterar dinamicamente seu conteúdo e propriedades. O JavaScript é uma das linguagens de programação mais populares e é amplamente usada em todo o mundo para o desenvolvimento de aplicativos web e móveis.Existe a linguagem php, que também,embora tenho sido mais, é muito usada

FRAMEWORKS

Drescrição
VOlTAR
Frameworks JavaScript são conjuntos de ferramentas e bibliotecas que ajudam os desenvolvedores a criar aplicativos web de forma mais fácil e eficiente. Existem muitos frameworks populares, como o React, Angular, Vue, Ember e muitos outros. Cada um desses frameworks tem suas próprias características e abordagens para o desenvolvimento de aplicativos web. Por exemplo, o React é uma biblioteca de interface de usuário que permite aos desenvolvedores criar componentes reutilizáveis ​​para construir interfaces de usuário eficientes. Já o Angular é um framework completo para o desenvolvimento de aplicativos web que inclui ferramentas para gerenciamento de estado, injeção de dependência e outras funcionalidades. Cada um desses frameworks oferece uma maneira única de criar aplicativos web e é importante para os desenvolvedores escolher o framework certo para o trabalho que precisam realizar.FRAMEWORKS FOCADOS EM WEB E JS
